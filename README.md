

# 🩺 Skin Disease Classification using Deep Learning

## 📌 Project Overview

This project focuses on automated **skin disease classification** using deep learning techniques. Multiple architectures including **CNN, EfficientNet, Vision Transformer (ViT), and XAI-based analysis** were implemented and compared to improve classification accuracy and model generalization.

The models were trained on a dataset of **9,000 dermoscopic skin images**.

---

## 🚀 Models Implemented

* Convolutional Neural Network (CNN)
* EfficientNet
* Vision Transformer (ViT)
* XAI (Explainable AI) for model interpretability

---

## 🗂 Dataset

* Total Images: 9,000
* Image preprocessing applied:

  * Resizing
  * Normalization
  * Data augmentation (rotation, flipping, etc.)
* Dataset split into training and validation sets

> ⚠️ Note: The dataset is not included in this repository due to size limitations.

## 💻 How to Run the Project

### Option 1: Run on Google Colab

1. Open the `.ipynb` notebook in Google Colab
2. Upload dataset to Colab or mount Google Drive
3. Run all cells

### Option 2: Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/Skin_Disease_Classification.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the notebook or training script.
